**DNS** - Domain Name System. 

![[dns.jpg]]

 **TLD** (top level domain) сервер. Домены верхнего уровня - это то что идет после последней точки _.com_, _.org_, _.ru_, _.net_. Кстати, существуют Generic Top Level Domain (**gTLD**), которые не привязаны к стране, например:  

-  _.edu_ (образование);
- _.org_ (организации и некоммерческие сайты);
- ._com_ (коммерческие веб сайты);
- .ai (организации, связанные с искусственным интеллектом) и много других;

Также есть Country Code Top Level Domain (**ccTLD**), которые привязаны к стране: 

- _.ru_ (Россия)
- _.ua_ (Украина)
- _.by_ (Беларусь)
- _.kz_ (Казахстан)
- _.us_ (США) 
- _.uk_ (Британия)   

В терминологии DNS существует три типа запросов:

- **Recursive (рекурсивный)** - это запрос формата: "Пришли мне IP-адрес сайта wiki.merionet.ru"
- **Iterative (итеративный)** - это запрос формата: "Пришли мне IP-адрес сайта wiki.merionet.ru либо авторитативный DNS сервер"
- **Inverse (обратный) -** спрашивает все наоборот: "Какое доменное имя у такого-то IP - адреса?"
